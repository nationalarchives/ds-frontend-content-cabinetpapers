Technical requirements

This section provides information on which browsers will work best with our website, and advises you of settings that can be changed to optimise the website

Web browsers

This site has been designed for screen resolutions of 800 x 600 or greater, and has been optimised for use on Microsoft Internet Explorer 6 and above, and Mozilla Firefox 1.0 and above on the PC, and Safari 2.0.4 on the Mac.
The latest version of Internet Explorer is available to download from the Microsoft website.

The latest version of Firefox is available to download from the Mozilla website.

Flash Player

Flash Player 7 is required to view some of the rich media interactive content contained on the site. You can download this free of charge from the Adobe website.

PDF documents

This site uses Adobe Portable Document Format (PDF) documents.

Adobe Acrobat Reader is required to view or print PDF documents and can be downloaded free of charge.

You can convert PDF documents to text using Adobe's PDF web-based conversion service.

Adobe also provides tools to help blind and visually impaired users to access PDF documents.
